# Reports

최종 대외 제출용 보고서와 요약 MD를 보관합니다.

이 폴더는 “최종 제출물 확인용”입니다. 홈페이지를 만들 때는 `data/processed/`의 JSON/CSV를 직접 사용하고, 보고서 문구나 방법론 설명을 확인할 때 이 폴더의 DOCX를 참고하면 됩니다.

| 파일 | 설명 |
| --- | --- |
| `seoul_economic_policy_news_external_report.docx` | 대외 제출용 본문 보고서입니다. 계약 과업, 방법론, 결과표, 그림이 포함됩니다. |
| `seoul_economic_policy_news_external_report.md` | DOCX의 구성과 핵심 판단을 빠르게 확인하기 위한 요약 메타 문서입니다. |

웹 제작 시 상세 방법론, 표기 문구, 산출물 범위는 DOCX를 기준으로 확인하세요.

DOCX 본문에는 NLP, CAR, r, 정합, 그레인저 같은 용어 설명을 장별 문맥 안에 풀어두었습니다. 보조 설명이 더 필요하면 [`../docs/plain-language-guide.md`](../docs/plain-language-guide.md)를 참고하면 됩니다.

보고서 결과를 홈페이지 문구로 옮길 때는 “가능/구현/납품” 메시지와 “정합 강도 구분”을 함께 유지해야 합니다. 즉, 계약 과업은 웹 산출물로 구현 가능하다고 분명히 말하되, `r < 0.4` 정책을 강한 정합처럼 표현하면 안 됩니다.
