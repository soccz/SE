# Assets

`figures/`에는 보고서와 README에서 참조할 수 있는 결과 그림이 들어 있습니다.

이 폴더의 그림은 최종 홈페이지에서 반드시 그대로 써야 하는 원본 화면이 아닙니다. 분석 결과를 빠르게 확인하고, 웹 차트를 만들 때 어떤 구조와 색상으로 보여줄지 참고하기 위한 결과 이미지입니다.

처음 보는 사람은 아래처럼 이해하면 됩니다.

| 그림 종류 | 쉽게 말하면 |
| --- | --- |
| KPI 요약 그림 | 전체 분석 규모와 핵심 결과를 한눈에 보여주는 그림입니다. |
| 섹터 요약 그림 | 5개 섹터의 평균 r과 범위를 비교하는 그림입니다. |
| 정책별 r 분포 그림 | 50개 정책이 강한/높은/보통/약한 정합 중 어디에 속하는지 보여주는 그림입니다. |
| 섹터별 정책 판정 그림 | 각 섹터 안의 10개 정책을 같은 형식으로 비교하는 그림입니다. |
| 대표 정책 이중축 차트 | 특정 정책의 NLP 흐름과 CAR 흐름을 함께 보여주는 예시 차트입니다. |

홈페이지에서는 원본 PNG를 그대로 쓰기보다, `data/processed`의 JSON/CSV로 차트를 다시 그리는 방식을 권장합니다.

그 이유는 다음과 같습니다.

1. 사용자가 섹터나 정책을 클릭하면 차트가 즉시 바뀌어야 합니다.
2. 마우스오버 툴팁, 필터, 정렬 같은 인터랙션은 PNG만으로 구현하기 어렵습니다.
3. 낮은 r 정책은 화면에서 별도 배지와 설명을 붙여야 합니다.
4. 모바일 화면에서는 차트 크기와 범례 위치를 다시 잡아야 합니다.

따라서 이 폴더의 그림은 “결과 검수와 디자인 참고용”으로 보고, 실제 웹 데이터는 [`../data/processed/`](../data/processed/)를 기준으로 연결하세요.
