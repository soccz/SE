# Reference

기존 단일 HTML 시연본을 참고용으로 보관합니다.

이 폴더의 HTML은 “최종 홈페이지”가 아니라 “이런 방식으로 정책별 NLP/CAR 흐름을 보여줄 수 있다”는 참고본입니다. 구조와 아이디어는 참고하되, 신규 홈페이지는 별도 프론트엔드에서 재구현하는 편이 좋습니다.

| 파일 | 설명 |
| --- | --- |
| `policy_nlp_car_demo.html` | 50개 정책별 NLP 흐름과 CAR 정합 시연 HTML |

신규 홈페이지는 이 HTML을 그대로 배포하기보다, `data/processed/web_payload.json`을 기준으로 재구현하는 방식을 권장합니다.

처음 보는 사람이 이 HTML에서 확인할 것은 다음입니다.

1. 정책을 선택하면 NLP 흐름과 CAR 흐름을 함께 볼 수 있다는 점
2. 정책별로 r, lag, 선택 주식그룹이 달라진다는 점
3. 같은 데이터도 카드, 탭, 상세 차트로 다시 구성할 수 있다는 점

반대로 이 HTML을 그대로 최종본으로 쓰지 않는 이유는 다음입니다.

1. README와 문서에서 정리한 초심자용 설명이 충분히 들어가 있지 않습니다.
2. `약한 정합` 정책을 낮은 강도로 구분하는 화면 장치가 더 필요합니다.
3. 향후 디자인, 모바일 대응, 데이터 다운로드, 방법론 탭을 붙이려면 구조화된 프론트엔드가 더 적합합니다.
