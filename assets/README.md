| `fig07_policy_alignment_*.png` | 섹터별 정책 10개 정합 판정 |

| `fig_sector_panel_2_finance.png` | 금융/증권 10개 정책 형태 비교 |
| `fig_sector_panel_3_bio.png` | 바이오/제약 10개 정책 형태 비교 |
| `fig_sector_panel_4_realestate.png` | 부동산/건설 10개 정책 형태 비교 |
