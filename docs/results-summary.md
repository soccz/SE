# Results Summary

본 문서는 홈페이지에 표시할 수 있는 핵심 분석 결과를 정리합니다.

처음 보는 사람은 이 문서를 “결과 숫자 목록”이 아니라 “홈페이지 첫 화면에 어떤 메시지를 올릴지 정하는 기준표”로 보면 됩니다. 각 숫자는 독립적으로 쓰기보다 함께 보여줘야 오해가 줄어듭니다.

예를 들어 `방향 정합 50/50`은 중요한 결과지만, 이것만 크게 보여주면 “50개 정책 모두 강하게 맞았다”로 오해될 수 있습니다. 그래서 `평균 r=0.706`, `강한·높은 정합 38/50`, `약한 정합 4/50`을 같이 보여주는 구성이 좋습니다.

## 핵심 KPI

| 항목 | 값 |
| --- | ---: |
| 정책 수 | 50 |
| 섹터 수 | 5 |
| 섹터별 정책 수 | 10 |
| 정책 관련 뉴스 | 121,954건 |
| LLM 점수 가용 기사 | 118,602건 |
| 차트 시계열 | 7,139행 |
| 정책별 방향 정합 | 50/50 |
| 평균 정합 상관계수 | 0.706 |

### KPI를 쉽게 읽는 법

| KPI | 쉬운 해석 | 화면 문구로 쓸 때 |
| --- | --- | --- |
| 정책 수 50 | 분석 대상 정책이 50개라는 뜻입니다. | “50개 주요 경제정책 분석” |
| 섹터 수 5 | 정책을 5개 산업 묶음으로 나눴다는 뜻입니다. | “5개 섹터별 정책 흐름 비교” |
| 정책 관련 뉴스 121,954건 | 정책 전후 분석에 사용한 뉴스 규모입니다. | “정책 관련 뉴스 12만 건 기반” |
| LLM 점수 가용 기사 118,602건 | NLP 점수화가 가능한 기사 수입니다. | “LLM 점수 가용 기사 11.8만 건” |
| 차트 시계열 7,139행 | 정책 상세 차트를 그릴 수 있는 날짜별 데이터 행 수입니다. | “정책별 상세 라인 차트 제공” |
| 방향 정합 50/50 | 50개 정책 모두에서 뉴스 흐름과 CAR 흐름의 방향이 같은 쪽으로 잡혔다는 뜻입니다. | “전 정책 방향 정합 확인” |
| 평균 r 0.706 | 전체 정책의 평균 정합 강도가 비교적 높은 편이라는 뜻입니다. | “평균 정합 상관계수 r=0.706” |

주의할 점은 KPI가 “성과 홍보 문구”가 아니라 “분석 결과를 읽는 입구”라는 점입니다. 숫자를 크게 보여주되, 바로 아래에 정합 강도 분포와 약한 정합 사례로 이어지게 구성하는 편이 안전합니다.

## 섹터별 요약

### 섹터별 표를 쉽게 읽는 법

이 표는 “어느 섹터가 평균적으로 더 잘 맞았는가”와 “섹터 안에 약한 사례가 있는가”를 함께 보여줍니다.

| 컬럼 | 쉬운 의미 |
| --- | --- |
| 정책 수 | 해당 섹터에 들어 있는 정책 개수입니다. 현재는 모든 섹터가 10개입니다. |
| 기사 수 | 해당 섹터 정책들과 관련된 뉴스 규모입니다. 기사 수가 많으면 보도량이 컸다는 뜻입니다. |
| 평균 r | 섹터 안의 10개 정책 정합도를 평균낸 값입니다. |
| 최소 r | 섹터 안에서 가장 약한 정책의 r입니다. 약한 사례를 찾을 때 중요합니다. |
| 최대 r | 섹터 안에서 가장 강한 정책의 r입니다. 대표 사례를 찾을 때 유용합니다. |

예를 들어 바이오/제약은 평균 r이 0.663으로 낮지 않지만, 최소 r이 0.069입니다. 즉 섹터 전체가 약하다는 뜻이 아니라, 섹터 안에 매우 약한 정책이 포함되어 있다는 뜻입니다. 2차전지/에너지도 평균은 0.709이지만 최소 r이 0.183이므로 약한 정책을 별도로 표시해야 합니다.

## 정책별 정합 등급 분포

| 등급 | 기준 | 정책 수 |
| --- | --- | ---: |
| 강한 정합 | r >= 0.8 | 23 |
| 높은 정합 | 0.6 <= r < 0.8 | 15 |
| 보통 정합 | 0.4 <= r < 0.6 | 8 |
| 약한 정합 | r < 0.4 | 4 |

정책별 판정은 [`data/processed/policy_alignment_status.csv`](../data/processed/policy_alignment_status.csv)를 기준으로 합니다. `r < 0.4` 정책은 방향은 같지만 강한 정합 사례로 보지 않습니다.

### 등급 분포를 쉽게 읽는 법

50개 정책을 `r` 크기에 따라 네 묶음으로 나눴습니다.

| 묶음 | 해석 |
| --- | --- |
| 강한 정합 23개 | 뉴스 흐름과 CAR 흐름이 매우 비슷하게 움직인 정책입니다. 홈페이지의 대표 사례로 쓰기 좋습니다. |
| 높은 정합 15개 | 강한 사례만큼은 아니지만 전반적으로 흐름이 잘 맞는 정책입니다. |
| 보통 정합 8개 | 방향은 맞고 일정한 정합은 있으나, 강한 사례처럼 강조하기는 어렵습니다. |
| 약한 정합 4개 | 방향은 같지만 r이 낮습니다. 반드시 낮은 강도 표시가 필요합니다. |

따라서 대외 문구는 “50개 정책 모두 방향 정합을 확인했고, 이 중 38개는 강한·높은 정합을 보였다”처럼 쓰는 것이 좋습니다.

최고 정책은 “대표적으로 흐름이 잘 맞은 사례”로 사용할 수 있습니다. 최저 정책은 “분석 결과를 과장하지 않고 약한 사례도 함께 공개한다”는 신뢰 장치로 사용할 수 있습니다.

## 약한 정합 정책

아래 4개 정책은 방향 정합은 유지되지만 `r < 0.4`이므로 웹에서는 회색 또는 낮은 강도 배지로 표시하는 것을 권장합니다.

| 정책 | 섹터 | r | 선택 주식그룹 |
| --- | --- | ---: | --- |
| P33 문케어 보장성 강화대책 | 바이오/제약 | 0.0690 | KOSPI·대기업 |
| P36 코로나19 예방접종 개시 | 바이오/제약 | 0.1221 | KOSDAQ·대기업 |
| P50 그린뉴딜 | 2차전지/에너지 | 0.1834 | KOSDAQ·소기업 |
| P46 미국 IRA | 2차전지/에너지 | 0.3057 | KOSPI·대기업 |

이 정책들은 삭제하거나 숨길 대상이 아닙니다. 오히려 보여주는 것이 좋습니다. 다만 “정합 확인”이라는 한 단어만 쓰면 강한 사례처럼 보일 수 있으므로, 카드와 차트에 다음처럼 표시합니다.

```text
방향 정합 · 약한 정합 · r=0.069
```

이 표현은 두 가지를 동시에 말합니다.

1. 방향은 같은 쪽으로 잡혔다.
2. 하지만 강도는 낮으므로 강한 정합 사례로 해석하면 안 된다.

## 웹에서 강조할 문구

- 정책 관련 뉴스의 NLP 반응흐름이 정책주 초과수익과 섹터별로 다르게 정합했다.
- 50개 정책 각각에 대해 정책 단위의 정합 판정을 제공한다.
- 낮은 r 정책은 방향 정합과 정합 강도를 분리해 표시한다.
- 정책 뉴스는 반도체, 금융/증권, 바이오/제약, 부동산/건설, 2차전지/에너지에서 서로 다른 방식으로 시장에 읽혔다.
- NLP 반응, CAR 정합, 그레인저 선행성 진단을 결합해 정책 뉴스와 시장 반응의 흐름을 입체적으로 점검했다.

## 홈페이지 첫 화면용 쉬운 요약 문구

아래 문구는 결과를 처음 보는 독자에게 적합한 톤입니다.

```text
이 프로젝트는 50개 주요 경제정책을 대상으로 정책 관련 뉴스의 흐름과 관련 주식그룹의 초과수익 흐름을 비교했다. 전체 정책에서 방향 정합이 확인됐고, 평균 정합 상관계수는 r=0.706이었다. 다만 모든 정책이 강하게 맞은 것은 아니므로, 정책별 r 등급을 함께 제공한다.
```

약한 사례 안내는 다음처럼 붙이면 됩니다.

```text
P33, P36, P46, P50은 방향은 같지만 r 값이 낮아 약한 정합으로 분류된다. 이 정책들은 낮은 강도 배지와 함께 표시해 결과 해석이 과장되지 않도록 한다.
```
