# Methodology

본 문서는 웹 페이지의 방법론 섹션과 데이터 설명 팝업에 사용할 분석 절차를 정리합니다.

## 아주 쉬운 설명

이 분석은 정책 뉴스를 “숫자로 된 흐름”으로 바꾼 뒤, 그 흐름이 관련 주식의 움직임과 비슷했는지 비교한 것입니다.

예를 들어 어떤 정책 뉴스가 계속 긍정적으로 보도되고, 같은 기간 관련 주식그룹의 초과수익도 비슷한 방향으로 움직였다면 정합도가 높게 나옵니다. 반대로 뉴스 흐름과 주가 흐름이 크게 다르면 정합도가 낮게 나옵니다.

따라서 이 보고서의 핵심은 “정책 뉴스와 시장 반응이 함께 움직인 정도를 정책별로 보여준다”입니다. “뉴스가 주가를 반드시 움직였다”거나 “주가 예측모형을 만들었다”는 뜻은 아닙니다.

## 분석 대상

- 주요 경제정책 50건
- 5개 섹터: 반도체, 금융/증권, 바이오/제약, 부동산/건설, 2차전지/에너지
- 섹터별 정책 수: 10건
- 실제 분석 표본의 정책 기준일: 1998-06-29 ~ 2024-11-04
- 분석 창: 정책일 D0 기준 -30일부터 +30일까지

## NLP 신호

정책 관련 뉴스가 시장에서 어떻게 읽혔는지를 수치화했습니다. 대외 표기는 아래 신호명을 사용합니다.

| 섹터 | 신호명 | 코드 | lag |
| --- | --- | --- | ---: |
| 반도체 | LLM 시장반응 신호 | `llm_market_response` | 0 |
| 금융/증권 | LLM 시장반응 신호 | `llm_market_response` | 20 |
| 바이오/제약 | PTEI 정책수용 신호 | `ptei_policy_acceptance` | 10 |
| 부동산/건설 | 복합 정책반응 신호 | `llm_ptei_composite` | 0 |
| 2차전지/에너지 | LLM 정책전환 반응 신호 | `llm_transition_response` | 20 |

정책별 사후 부호 변경은 적용하지 않았습니다. 섹터별 신호 정의와 lag를 고정하고, 정책별 주식그룹은 기존 6분할 중 정합도가 가장 높은 그룹을 선택했습니다.

## 시장 반응과 CAR

정책 관련 주식그룹의 시장 반응은 초과수익률을 기준으로 계산했습니다.

1. KOSPI/KOSDAQ과 기업규모를 조합한 6개 주식그룹을 유지합니다.
2. 그룹 수익률에서 KOSPI 수익률을 차감해 초과수익률을 계산합니다.
3. 초과수익률을 누적해 CAR를 산출합니다.
4. 정책별로 NLP 누적흐름과 가장 잘 정합하는 주식그룹을 선택합니다.

## 정합 판정

정합 판정은 정책별 NLP 누적흐름과 선택 주식그룹 CAR의 상관계수 `r`을 기준으로 표시합니다.

| 등급 | 기준 |
| --- | --- |
| 강한 정합 | `r >= 0.8` |
| 높은 정합 | `0.6 <= r < 0.8` |
| 보통 정합 | `0.4 <= r < 0.6` |
| 약한 정합 | `r < 0.4` |

웹에서는 `alignment_status`, `alignment_grade`, `r`를 함께 표시하면 됩니다. `alignment_status`는 방향 판정이고, `alignment_grade`는 정합 강도입니다. 따라서 `r=0.069` 같은 정책은 방향은 맞지만 강도는 약한 사례로 해석해야 합니다.

## 그레인저 선행성 검정

계약서상 인과검정 항목에 대응해 그레인저 선행성 검정 체계를 구성했습니다.

- 누적형 NLP/CAR 계열은 시각화에 사용합니다.
- 검정에는 일별 원계열을 사용합니다.
- 종속변수는 일별 정책주 초과수익입니다.
- 설명변수는 일별 NLP 신호의 과거값입니다.
- 정책 고정효과와 자기과거수익 통제를 포함한 패널 검정을 병행했습니다.

웹 방법론 섹션에서는 “NLP 반응, CAR 정합, 그레인저 선행성 진단을 결합해 정책 뉴스와 시장 반응의 흐름을 점검했다”는 표현을 권장합니다.
